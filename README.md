# Snake - Push Boxes

This is a mini-game called Push Boxes, incorporating elements of the Snake game. The game's entry file is main.py.

Controls:
- Use the W, A, S, D keys or the arrow keys to move the snake.
- Use the left mouse button to click the game button.

Gameplay:
- The silver boxes are immovable.
- Players need to push the boxes past each checkpoint (a black rectangle with colored circular stripes) to eliminate that checkpoint. Once the box has passed all checkpoints, it needs to be pushed to the finish line (a colored rectangle with black square stripes). When the box eliminates all checkpoints and the finish line, it turns silver and cannot be moved.
- Once all boxes turn silver, the current level is completed, and you can choose whether to continue to the next level.
- If the checkpoints and finish lines are different colors, the boxes cannot be eliminated.
